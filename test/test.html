Possible steps to achieve:
      1) Initial: Wallet allows generation of any kind of transaction manually
      2) Request: app can request specific criteria or instructions for transaction
      3) Allow: User can allow wallet to attempt to complete those instructions
      4) Finalize: User can customize (ins, outs, fees) and finish the transaction creation
      5) Broadcast: If desired, can broadcast the transaction
      6) Management: can CPFP or RBF broadcast transactions
